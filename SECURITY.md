# Security Policy

## Reporting a Vulnerability

If you discover a security issue in DevPath Navigator, please report it
**privately** via GitHub's
[Private Vulnerability Reporting](https://github.com/muroshima/DevPath-Navigator/security/advisories/new)
(repo **Security** tab → "Report a vulnerability").

Please do not open a public issue or pull request describing the
vulnerability. We aim to acknowledge reports within ~7 days.

## Scope

This repository is a hackathon entry (DevOps × AI Agent Hackathon 2026)
and runs on **synthetic data only** — no real user / personnel data is
involved. The hosted demo deployment will be taken down after the
hackathon submission window.

Reports outside of the deployed services (e.g. dependency advisories
on transitive packages, theoretical issues without an exploit path) are
welcome but may be triaged at lower priority.
